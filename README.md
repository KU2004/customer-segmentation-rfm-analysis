# Customer Segmentation using RFM Model

## Project Overview

This project applies the RFM (Recency, Frequency, Monetary) framework to segment customers based on purchasing behavior.

---

## What is RFM?

### Recency
How recently the customer purchased.

### Frequency
How often the customer purchases.

### Monetary
How much money the customer spends.

---

## Business Problem

Businesses need to understand:

- High-value customers
- Loyal customers
- At-risk customers
- Lost customers

to improve retention and revenue.

---

## Objective

Create a Power BI dashboard for customer segmentation and CRM analytics.
